Santo Grial - Elementos Genéricos

Bienvenido al sitio web de demostración. Aquí utilizamos elementos genéricos como `div` para organizar el contenido de la página de manera flexible.

Navegación

Esta es la barra lateral izquierda, donde típicamente puedes incluir un menú de navegación, enlaces a otras secciones del sitio o widgets adicionales.

Contenido Principal

Este es el espacio principal de contenido. Aquí puedes agregar textos, imágenes, videos, tablas y cualquier otro tipo de información que quieras destacar.

Los elementos genéricos como el `div` son muy útiles para agrupar bloques de contenido y aplicarles estilos a través de CSS, permitiendo organizar la disposición de una página web de forma flexible.

Información Adicional

Esta es la barra lateral derecha, donde puedes colocar información relevante o complementaria como un boletín de noticias, anuncios o enlaces relacionados.
